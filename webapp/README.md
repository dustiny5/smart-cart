# Webapp

## Setup

```bash
# Inside the ./webapp directory
npm i
npm run dev
```

## Dependencies

-   React 19
-   Tailwindcss 4.1

## Assets

-   [Heroicons](https://heroicons.com/)

## Deploy

-   Project name, `smart-cart-webapp`, and hosted on [Netlify](https://app.netlify.com/)
-   Configure a `netlify.toml` and deployed the webapp
